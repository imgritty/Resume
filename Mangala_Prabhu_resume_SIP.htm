Projects:

The MegaHub MM is a Media Gateway Controller (soft switch) designed to make the best of, and bridge the gap between the MegaHub 600E call model (the top tandem switch in the North American market, with its extensive feature set) and next-generation network components (gateways and soft switches) providing the feature, density, and cost savings those components are anticipated to achieve.

The MCI MM is an MGC and supports two of these protocols Media Gateway Control Protocol (MGCP) /Media Gateway Protocol (MEGACO) for Media Control. MCI MM is a Next Generation Soft Switch, which is ported from MegaHub 600E. The MegaHub 600E currently is a long distance tandem switch capable of handling telephony calls in a circuit-switched network. The MCI-WCOM MM is an initial step towards integration of the traditional circuit switched network and the next-generation packet-based network.
